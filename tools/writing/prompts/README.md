# Writing Prompts

Store reusable drafting prompt fragments here.

Guidelines:
- Keep drafting instructions story-neutral and in English.
- Use placeholders for story title, language, chapter number, canon, brief, and prior summary.
- Do not include active manuscript text or private canon.
- Short multilingual examples are allowed only to demonstrate language routing.

## MVP Draft Prompt Fragment

Use this fragment when a writing command needs a compact instruction block:

```text
Write chapter {chapter_number} for {story_title} in {language}.
Use only the supplied write pack, canon, reveal lock, and prior context.
Do not add author notes, explanations, hidden canon, or reviewer guidance.
Return only publishable prose for review.
```
