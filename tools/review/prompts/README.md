# Review Prompts

Store reusable reviewer prompt fragments here.

Guidelines:
- Keep reviewer instructions story-neutral and in English.
- Ask reviewers to use supplied canon, briefs, and draft extracts only.
- Do not encode a specific cast, setting, mystery, or spoiler sequence.
- Short multilingual examples are allowed only to show language-aware output.

## MVP Review Prompt Fragment

Use this fragment when a review command needs a compact instruction block:

```text
Review chapter {chapter_number} for {story_title}.
Use only the supplied review pack, draft, canon, reveal lock, and reviewer profile.
Identify blocker and major issues before minor polish.
Return the standard review report format with gate guidance.
```
