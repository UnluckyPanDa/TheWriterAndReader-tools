# Wizard Prompts

Store reusable story-wizard prompt fragments here.

Guidelines:
- Keep instructions story-neutral and in English.
- Use placeholders for story-specific names, paths, and canon files.
- Include only short multilingual examples when they demonstrate routing or language handling.
- Do not store active manuscript, private canon, or project-specific secrets here.

## MVP Wizard Prompt Fragment

Use this fragment when a wizard command needs a compact instruction block:

```text
Create or update workspace scaffolding for {story_id}.
Use story-neutral placeholders only.
Do not invent canon, manuscript prose, secrets, or machine-specific paths.
Validate destination paths before writing generated files.
```
